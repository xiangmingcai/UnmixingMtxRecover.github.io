Unmixing Matrix Recover (UMR)

(version 1.0)

This website is designed to guide users recover the unmixing matrix for flow cytometry analysis.

With the UMR, you could recover your unmixing matrix from fcs file with both raw signal and unmixed signal.

However, keep in mind, the recovered unmixing matrix could be significantly different from the original matrix, because of rounded number and potential different unmixing matri used.

If you find this tool helpful, please remember to cite us in your work. View this project on GitHub
